🎓 Student Management System

A web-based Student Management System built using Java (JSP, Servlet, JDBC) and MySQL.
This project demonstrates complete CRUD operations and basic MVC architecture.

---

🚀 Features

🔐 Login System
➕ Add Student
📋 View Students
✏️ Update Student
🗑️ Delete Student

---

🛠️ Tech Stack

Frontend: HTML, CSS, JSP
Backend: Java Servlets, JDBC
Database: MySQL
Server: Apache Tomcat

⚙️ How to Run

Clone the repository
Import into Eclipse / IntelliJ
Configure Apache Tomcat

Create database:
CREATE DATABASE studentcrm;

Create table:
CREATE TABLE srecord(
name VARCHAR(50),
email VARCHAR(50),
mobile VARCHAR(15)
);

Update DB credentials in:
DBServiceImpl.java

Run project on server
